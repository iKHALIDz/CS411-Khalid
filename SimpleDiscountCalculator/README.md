Simple Discount Calculator As an iPhone applicationBrief Description:This iPhone application enables the user to calculate the discount price of an item based on the entered original price, tax, discount rate and amount of dollars. The user is able to swipe to the left to see a graphical comparison of the original price and the discounted price, and an additional swipe to will return the user to main view.Copyright:This is app is an assignment for CS411 course, designed and implemented by Khalid Alahmari â€“ CWID: 893-11-9289
