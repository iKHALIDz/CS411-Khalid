Simple Calculator As an iPhone applicationBrief Description:This iPhone application enables the user to to simple binary Calculation including addition, subtraction, multiplication, and division between two real numbers. Also is supported unary operation such as changing the sign and the calculate the square root for a number. Also The decimal point is supported by this app.All the minimum required features are complete.
